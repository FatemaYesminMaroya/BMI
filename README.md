# BMI Calculator 

The **BMI (Body Mass Index) Calculator** is a simple and interactive web application built using **HTML, CSS, and JavaScript**. It allows users to quickly calculate their BMI based on their height and weight and provides an easy-to-understand health classification.

---

## Features

- **User-Friendly Interface:** Clean and responsive design for smooth use on desktop and mobile.  
- **Real-Time Calculation:** Calculates BMI instantly as the user enters height and weight.  
- **Health Categories:** Shows BMI result with health category: Underweight, Normal, Overweight, or Obese.  
- **Responsive Design:** Works across all screen sizes.  
- **Interactive Experience:** Provides instant feedback without page reloads.
